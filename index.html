ملاحظات ونصائح أثناء تعلم HTML

تحذير: لا تستخدم الوسوم القديمة مثل <font> أو <center> — استخدم CSS بدلاً منها.
نصيحة: حاول استخدام الوسوم الدلالية (semantic) لتحسين وصولية الصفحة وSEO.
معلومة: ضع نص بديل alt للصور دائماً — يساعد المستخدمين ومحركات البحث.
ملاحظة: اجعل الأكواد داخل <code> و <pre> ووفّر زر نسخ لتجربة سهلة.

مقدمة عن HTML

  • يعني إيه HTML؟
    HTML هي اللغة اللي بتخلي أي صفحة ويب تظهر على الإنترنت. زي الأساس اللي بتبني عليه بيتك، كل موقع لازم يبدأ بـ HTML.
  • ليه أتعلم HTML؟
    لو عاوز تصمم موقع أو حتى تعدل على صفحة، لازم تكون فاهم HTML. هي البداية لأي شغل ويب، حتى لو هتتعلم لغات تانية بعد كده.
  • إزاي بتشتغل؟
    بتكتب وسوم (Tags) زي <h1> أو <p>، كل وسم ليه وظيفة معينة. المتصفح بيقرأ الوسوم دي ويعرض الصفحة بشكل منظم.
  • مثال عملي:
    
<h1>أهلاً بيك في أول درس HTML</h1>
<p>النهاردة هنتعلم إزاي نكتب صفحة ويب بسيطة.</p>

    أهلاً بيك في أول درس HTML

    النهاردة هنتعلم إزاي نكتب صفحة ويب بسيطة.

  • نصيحة:
    جرب تكتب بنفسك كل مثال، حتى لو بسيط. الممارسة هي اللي هتخليك تتعلم بسرعة.

الوسوم الأساسية في HTML

  • <!DOCTYPE html>
    معناه: بيعرف المتصفح إن الصفحة دي HTML5 (أحدث إصدار).
    ليه مهم؟ عشان المتصفح يقرأ الصفحة صح ويعرضها بشكل حديث.
  • <html> ... </html>
    معناه: بداية ونهاية صفحة الويب. كل حاجة بتكتبها لازم تكون جوا الوسم ده.
  • <head> ... </head>
    معناه: جزء خاص بالمعلومات اللي مش بتظهر للمستخدم (زي العنوان، وصف الصفحة، روابط ملفات خارجية).
  • <body> ... </body>
    معناه: كل اللي المستخدم بيشوفه في الصفحة (نصوص، صور، جداول... إلخ).
  • <h1> ... <h6>
    معناه: وسوم العناوين. <h1> أكبر عنوان، <h6> أصغر عنوان.

    هذا عنوان رئيسي

    عنوان فرعي

    عنوان أصغر

    
<h1>هذا عنوان رئيسي</h1>
<h2>عنوان فرعي</h2>
<h3>عنوان أصغر</h3>
  • <p>
    معناه: فقرة نصية. أي شرح أو جملة بتحطها جوا الوسم ده.

    هذا مثال على فقرة نصية تشرح فكرة معينة أو تقدم معلومة.

    
<p>هذا مثال على فقرة نصية تشرح فكرة معينة أو تقدم معلومة.</p>
  • <img src="..." alt="...">
    معناه: لإضافة صورة.
    src: مسار الصورة (فين الصورة موجودة؟)
    alt: نص بديل يظهر لو الصورة مش ظهرت (مهم جداً لذوي الاحتياجات الخاصة ولسيو الموقع). 
    
<img src="https://via.placeholder.com/150x80?text=مثال+صورة" alt="مثال صورة">
  • <a href="..." target="_blank"> ... </a>
    معناه: لإضافة رابط.
    href: عنوان الرابط اللي هيروح له المستخدم.
    target="_blank": يخلي الرابط يفتح في صفحة جديدة (مهم لو مش عاوز المستخدم يسيب موقعك).
    اضغط هنا لزيارة موقع خارجي 
    
<a href="#" target="_blank">اضغط هنا لزيارة موقع خارجي</a>
  • <ul> ... </ul>
    معناه: قائمة غير مرتبة (نقط).
    • تفاح
    • موز
    • برتقال
    
<ul>
  <li>تفاح</li>
  <li>موز</li>
  <li>برتقال</li>
</ul>
  • <ol> ... </ol>
    معناه: قائمة مرتبة (أرقام أو حروف).
    type: يحدد نوع الترقيم (رقم، حرف كبير، حرف صغير... إلخ).
    start: يحدد رقم البداية في القائمة.
    1. خطوة أولى
    2. خطوة ثانية
    3. خطوة ثالثة
    
<ol type="A" start="2">
  <li>خطوة أولى</li>
  <li>خطوة ثانية</li>
  <li>خطوة ثالثة</li>
</ol>
  • <table> ... </table>
    معناه: جدول بيانات.
    جدول درجات الطلاب
    الاسم الدرجة
    روني 93
    محمد 90 
    
<table>
  <caption>جدول درجات الطلاب</caption>
  <thead>
    <tr>
      <th>الاسم</th>
      <th>الدرجة</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>روني</td>
      <td>93</td>
    </tr>
    <tr>
      <td>محمد</td>
      <td>90</td>
    </tr>
  </tbody>
</table>
  • <form> ... </form>
    معناه: نموذج لإدخال بيانات من المستخدم. 
    
<form>
  <input type="text" placeholder="اسم المستخدم" required>
  <input type="password" placeholder="كلمة المرور" required>
  <button type="submit">تسجيل الدخول</button>
</form>
  • <button type="..."> ... </button>
    معناه: زرار للتفاعل مع النموذج أو الصفحة.
    type: يحدد وظيفة الزر (submit لإرسال النموذج، button لزر عادي). 
    
<button type="button">اضغط هنا</button>

نصيحة: كل وسم ليه بداية ونهاية ماعدا بعض الوسوم زي <img> و <br> اللي بتتقفل في نفس السطر.

شرح أهم وسوم تنظيم الصفحة في HTML

  • <header>
    رأس الصفحة، غالباً فيه اسم الموقع أو اللوجو وقائمة التنقل.

    اسم الموقع

    • الرئيسية
    • دروس
    • تواصل
    
<header>
  <h1>اسم الموقع</h1>
  <nav>
    <ul>
      <li><a href="#">الرئيسية</a></li>
      <li><a href="#">دروس</a></li>
      <li><a href="#">تواصل</a></li>
    </ul>
  </nav>
</header>
  • <nav>
    قائمة التنقل بين صفحات أو أقسام الموقع.
    • الرئيسية
    • دروس
    • تواصل
    
<nav>
  <ul>
    <li><a href="#">الرئيسية</a></li>
    <li><a href="#">دروس</a></li>
    <li><a href="#">تواصل</a></li>
  </ul>
</nav>
  • <footer>
    فوتر الصفحة، غالباً فيه حقوق النشر وروابط التواصل.

    جميع الحقوق محفوظة © 2025

    • فيسبوك
    • تويتر
    
<footer>
  <p>جميع الحقوق محفوظة © 2025</p>
  <ul>
    <li><a href="#">فيسبوك</a></li>
    <li><a href="#">تويتر</a></li>
  </ul>
</footer>
  • <aside>
    جزء جانبي في الصفحة (يمين أو شمال)، غالباً فيه روابط أو معلومات إضافية.

    روابط مهمة

    • رابط 1
    • رابط 2
    
<aside>
  <h2>روابط مهمة</h2>
  <ul>
    <li><a href="#">رابط 1</a></li>
    <li><a href="#">رابط 2</a></li>
  </ul>
</aside>
  • <section>
    جزء من الصفحة مخصص لموضوع معين (مثلاً: مقدمة، دروس، أمثلة).

    مقدمة عن الموقع

    هنا تلاقي شرح سريع عن محتوى الموقع.

    
<section>
  <h2>مقدمة عن الموقع</h2>
  <p>هنا تلاقي شرح سريع عن محتوى الموقع.</p>
</section>
  • <div>
    حاوية عامة لأي عناصر، بتستخدمها عشان تجمع عناصر وتتحكم في تصميمهم.

    هذا صندوق عام

    تقدر تحط فيه أي عناصر وتغير شكله بالتصميم.

    
<div>
  <h3>هذا صندوق عام</h3>
  <p>تقدر تحط فيه أي عناصر وتغير شكله بالتصميم.</p>
</div>

وسوم HTML حديثة ومميزات إضافية

  • <main>
    يحدد المحتوى الرئيسي للصفحة، مفيد لمحركات البحث والوصولية.

    المحتوى الرئيسي هنا

    كل ما يهم الزائر يكون داخل هذا الوسم.

    
<main>
  <h2>المحتوى الرئيسي هنا</h2>
  <p>كل ما يهم الزائر يكون داخل هذا الوسم.</p>
</main>
  • <article>
    يستخدم لكتابة مقال أو محتوى مستقل داخل الصفحة.

    مقال تعليمي عن HTML

    هذا المقال يشرح أساسيات HTML.

    
<article>
  <h3>مقال تعليمي عن HTML</h3>
  <p>هذا المقال يشرح أساسيات HTML.</p>
</article>
  • <figure> و <figcaption>
    لعرض صورة أو رسم مع تعليق توضيحي. 
    
<figure>
  <img src="..." alt="صورة توضيحية">
  <figcaption>هذه صورة توضيحية مع تعليق.</figcaption>
</figure>
  • <details> و <summary>
    لعرض معلومات قابلة للفتح والإغلاق (مثل الأسئلة الشائعة).
    ما هي HTML؟

    HTML هي لغة بناء صفحات الويب.

    
<details>
  <summary>ما هي HTML؟</summary>
  <p>HTML هي لغة بناء صفحات الويب.</p>
</details>
  • المساعد (Accessibility)
    استخدم aria-label أو alt مع الصور والروابط لتسهيل الاستخدام لذوي الاحتياجات الخاصة. 
    
<button aria-label=\"إرسال النموذج\">إرسال</button>
<img src=\"...\" alt=\"وصف الصورة\">
  • SEO (تحسين ظهور الموقع)
    استخدم وسوم <title> و <meta name=\"description\" ...> ووسوم العناوين بشكل منظم. 
    
<head>
  <title>تعلم HTML خطوة بخطوة</title>
  <meta name=\"description\" content=\"دليل شامل لتعلم HTML للمبتدئين\">
</head>
  • استيراد خطوط Google Fonts
    لجعل الموقع أجمل وأسهل قراءة. 
    
<link href=\"https://fonts.googleapis.com/css2?family=Cairo:wght@400;700&display=swap\" rel=\"stylesheet\">
  • استخدام أيقونات SVG
    لإضافة رموز وأيقونات بجودة عالية. 
    
<svg width=\"24\" height=\"24\" fill=\"none\" stroke=\"currentColor\">
  <circle cx=\"12\" cy=\"12\" r=\"10\" stroke-width=\"2\"/>
</svg>

أسئلة شائعة عن HTML


ما هي HTML؟

HTML هي لغة بناء صفحات الويب وتستخدم لكتابة هيكل الصفحة.


هل أحتاج لتعلم HTML قبل CSS وJS؟

اه. لأن HTML هي الأساس وكل شيء في CSS وJS يعتمد عليها.
